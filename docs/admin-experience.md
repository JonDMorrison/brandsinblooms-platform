# Admin Experience

## Overview
Admins are Brands In Blooms platform administrators who oversee all sites. They access a super admin dashboard for global management, performing actions on any site, viewing analytics, and ensuring platform health. The interface is simple yet powerful, with easy navigation to avoid overwhelming non-technical admins.

## Key Features
- **Global Site Dashboard**: View all sites, with search, filter, and status indicators.
- **Site Management**: Impersonate or directly manage any site's content and products, similar to Site Owner but across all tenants.
- **Analytics and Engagement**: Overview of site traffic, content performance, and user engagement metrics.
- **User Management**: Handle site owner accounts, permissions.
- **Platform Tools**: AI generation for sites, manual product catalog management, system settings.
- **Product Catalog Management**: Oversee CSV/PDF import functionality and help site owners manage their in-store inventory displays.
- **Bloom Suite Integration**: Manage SSO setup and integration settings across the platform.

## User Journeys
### Daily Oversight
1. Log in to super admin dashboard.
2. View overview of all sites and recent activity.
3. Search for a specific site and manage its content or product catalog.

### Troubleshooting
1. Select a site from list.
2. Perform CRUD on pages/products as needed.
3. Help site owners with product catalog imports or content issues.

### Product Catalog Support
1. Assist site owner with CSV/PDF product import.
2. Review and troubleshoot catalog display issues.
3. Help configure product categories and organization.

### Reporting
1. Navigate to analytics section.
2. View traffic metrics, popular content, and engagement stats.
3. Export reports on site performance and user activity.

## Navigation Structure
- Home: Global overview, quick stats, site list.
- Sites: Detailed list with search/filter, links to manage individual sites.
- Analytics: Traffic metrics, content performance, engagement tracking.
- Users: Site owner management.
- Product Catalogs: Overview of catalog imports and management tools.
- Bloom Suite: Integration settings and SSO management.
- Settings: Platform-wide configurations.

Design focuses on clean layouts, searchable lists, and one-click actions to manage multiple sites efficiently without technical hurdles.
