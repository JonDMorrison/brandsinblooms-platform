# Future Customer Features (Phase 2)

This document outlines customer-facing features that will be implemented in Phase 2 of the Brands in Blooms platform, enabling full e-commerce functionality.

## Overview

Phase 2 will transform the platform from a showcase website into a full e-commerce experience, allowing customers to purchase products online with delivery through dropshipping partners.

## Key Features (Phase 2)

### Online Shopping Experience
- **Dual Catalog System**: Browse both "Shop Online" (dropshipped) and "Shop In-Store" (local inventory) products
- **Shopping Cart**: Add multiple products, adjust quantities, save for later
- **Guest Checkout**: Purchase without creating an account
- **Registered Accounts**: Save addresses, view order history, track shipments

### Product Browsing Enhancements
- **Endless Aisle Catalog**: Access to thousands of dropshipped garden products
- **Advanced Filtering**: Filter by availability (online vs in-store), price, category, brand
- **Product Comparison**: Compare multiple products side-by-side
- **Wishlist Functionality**: Save products for future purchase

### Checkout and Payment
- **Secure Payment Processing**: Stripe integration for credit/debit cards
- **Multiple Payment Methods**: Credit cards, debit cards, digital wallets
- **Shipping Calculations**: Real-time shipping quotes based on location
- **Tax Calculations**: Automatic tax calculation by jurisdiction
- **Order Review**: Clear summary before purchase confirmation

### Post-Purchase Experience
- **Order Confirmation**: Immediate email confirmation with order details
- **Order Tracking**: Real-time shipment tracking
- **Delivery Updates**: Email/SMS notifications for order status changes
- **Digital Receipts**: Downloadable invoices and receipts
- **Easy Returns**: Self-service return initiation

### Account Management
- **Order History**: View all past purchases
- **Reorder Functionality**: Quickly reorder previous items
- **Address Book**: Save multiple shipping addresses
- **Payment Methods**: Securely store payment information
- **Communication Preferences**: Manage email and SMS notifications

## User Journeys (Phase 2)

### Online Purchase Journey
1. Browse products marked as "Available Online"
2. Read detailed descriptions and reviews
3. Add items to cart
4. Proceed to checkout
5. Enter shipping and payment information
6. Review and confirm order
7. Receive confirmation and tracking information

### Hybrid Shopping Journey
1. Browse both online and in-store catalogs
2. Add online items to cart for delivery
3. Note in-store items for pickup
4. Complete online purchase
5. Visit store for in-store items

### Account-Based Shopping
1. Log into account
2. View personalized recommendations
3. Access saved addresses and payment methods
4. Quick reorder from purchase history
5. Track all active orders in one place

## Technical Features (Phase 2)

### E-commerce Infrastructure
- **Inventory Sync**: Real-time availability from dropshipping partners
- **Payment Security**: PCI-compliant payment processing
- **Order Management**: Automated order routing to suppliers
- **Multi-warehouse Support**: Optimal shipping from closest location

### Enhanced User Features
- **Product Reviews**: Customer ratings and reviews
- **Gift Options**: Gift messages and special packaging
- **Promotional Codes**: Discount and coupon support
- **Abandoned Cart Recovery**: Email reminders for incomplete purchases

### Mobile Commerce
- **Mobile-Optimized Checkout**: Streamlined mobile purchase flow
- **Apple Pay/Google Pay**: One-touch payment options
- **Mobile Order Tracking**: Track shipments on the go
